# AM92-Calculator
ENGLISH VERSION
A simple AM92 Select Calculator according to the newest IFoA Life Table.

Known problems:
1. Can only calculate whose life is an interger.
2. A person's q_[x] is unknown and it's estimated according to q_[x-t] + t for whose age is above 90.
3. Can only calculate according to Life table AM92.

Please tell me if there is a new bug.
